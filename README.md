# Priority-Timetable

Timetable is a fast, elegant, and fully private personal productivity planner built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no server required. Inspired by the philosophy that great tools should be simple, offline-first, and entirely yours.
The design draws from editorial aesthetics, pairing warm typographic choices with a rich glass-morphism interface layered over a cinematic background — built to feel as considered as the work it helps you manage.

Features & Functionality

⬡ Priority Page
The heart of the app. All active tasks are organised into three collapsible bands — High, Medium, and Low priority — each visually distinct with colour-coded indicators and deadline badges. Tasks can be dragged and dropped between priority levels, edited inline by clicking the task name, broken into sub-tasks with individual checkboxes and auto-calculated progress bars, and filtered by urgency (Overdue, Today, This Week). A live statistics bar at the top gives you an instant overview of what needs attention.

◫ Calendar Page
A fully interactive monthly calendar where you can manually add events to any date. Each event carries a title, time, priority level, and optional note. Events appear as coloured chips directly on the calendar grid, and an Upcoming Events list below surfaces everything due in the next 30 days in chronological order. Navigate between months freely, or jump back to today in one click.

◎ Completed Page
A dedicated archive of every task you've finished, grouped by the date it was completed. From here you can review your history, filter completed tasks by priority level, restore any task back to active with a single click, or permanently clear your history. A completion rate stat gives you a running measure of your overall progress.
Across All Pages

Live clock in the sidebar, updating every second
Full-text search across task names, notes, and sub-tasks
Export & Import — back up your entire timetable as a .json file at any time, or import it on another device to pick up exactly where you left off
100% private — all data lives in your browser's localStorage. Nothing is ever sent to a server, no accounts, no tracking
